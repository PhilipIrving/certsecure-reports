CertSecure Data

Compliance sync & report generation — HUB International

Internal Use Only
Automated compliance data sync and report generation for HUB CertSecure clients. Pulls data nightly from the Evident API and Google Sheets, stores it as CSVs in the data/ folder, and generates branded Excel and PDF reports on demand via GitHub Actions.
Workflows
CertSecure Data Sync
Nightly — 9:00 PM CT
Runs sync.js → sync-engagement.js → sync-custom-properties.js and commits updated CSVs to data/.
Generate CertSecure Reports
Manual — GitHub Actions
Runs generate_reports.py for one client or all active clients, then commits Excel + PDF reports to reports/.

Reports
Compliance Summary
PDF
KPIs, compliance & COI status breakdown, no-COI breakdown, expiring coverage by 7 / 30 / 60 / 90 day windows.
Compliance Detail
Excel · 5 Sheets
Summary, Entity List, Expiring Coverage, NC by Coverage & Criteria, Non-Compliance Reasons.
Engagement Summary
PDF
Email KPIs, open/click rates, deliverability breakdown, delivery issue types, 60-day engagement funnel.
Engagement Detail
Excel · 5 Sheets
Summary, No COIs on File, No Engagement Since Go-Live, NC + No Recent Engagement, Undeliverable Emails.
Generating Reports
How to Run a Report
1
Open GitHub Actions
Go to the Actions tab in this repo.
2
Select the workflow
Click Generate CertSecure Reports in the left sidebar.
3
Run workflow
Click Run workflow (top right). Fill in the inputs — both are optional.
4
Inputs
Client name — leave blank for all active clients, or enter exact name e.g. Canadian Pacific Kansas City

Report types — leave blank for all 4, or comma-separated e.g. compliance_detail,engagement_detail
5
Download
Reports are committed to reports/ when done. Download directly from GitHub.

sync_metadata.csvGeneratedTracks when each CSV last changed and last ran
Custom Properties Whitelist
Whitelisted Fields — custom_properties.csv

Only these 10 fields are written to custom_properties.csv. All other Evident custom fields are ignored. To change the whitelist, edit the FIELD_WHITELIST constant in src/sync-custom-properties.js.

contractNumber notificationList entityType projectName projectNumber propertyLocation tenantType propertyManager insuranceType Latest Submission Date

Security
All API keys are stored as GitHub Secrets and never written to any file in this repo. Branch protection is enabled on main — direct pushes are blocked except from github-actions[bot]. Secret scanning is enabled. The data/ folder contains client compliance data — do not commit any file containing API keys, passwords, or personal data outside the data/ folder.
